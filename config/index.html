Example WODIN configuration

This demo site demonstrates how a WODIN instance can be configured. All configuration files are in the /config folder.

This index page can be found at index.html.

Each app is configured using an [appName].config.json file where [appName] is the route to the app under /apps.

Sample data can be provided in the /files sub-folder. The examples here have parent folders with same name as the app, but any folder structure can be used.
